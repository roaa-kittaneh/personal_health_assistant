# نتائج البحث عن مصادر البيانات الطبية

## المصادر الموثوقة المتاحة

### 1. MedlinePlus
**الموقع:** https://medlineplus.gov/
**الوصف:** مورد معلومات صحية عبر الإنترنت من المكتبة الوطنية للطب (NLM)، جزء من المعاهد الوطنية للصحة (NIH).
**المميزات:**
- معلومات عن أكثر من 1000 مرض وحالة صحية
- موضوعات منظمة حسب أنظمة الجسم والحالات والتشخيص والعلاج
- معلومات منتظمة ومحدثة يومياً
- متاح باللغة الإسبانية والإنجليزية
- يغطي: الأدوية والمكملات، الاختبارات الطبية، الموسوعة الطبية

### 2. PubMed Central (PMC)
**الموقع:** https://pmc.ncbi.nlm.nih.gov/
**الوصف:** أرشيف مفتوح الوصول للأدبيات الطبية والعلوم الحياة من NIH/NLM
**المميزات:**
- 11.5 مليون مقالة مؤرشفة
- نصوص كاملة لمقالات البحث الطبي
- واجهات برمجية (APIs) متعددة للوصول الآلي
- خدمات سحابية على AWS

### 3. واجهات البرمجيات المتاحة في PMC

| اسم الواجهة | الوصف | الرابط الأساسي |
|-----------|-------|------------|
| **OA API** | بيانات الاستشهادات ومعلومات الترخيص | https://www.ncbi.nlm.nih.gov/pmc/utils/oa/oa.fcgi |
| **PMC OAI-PMH** | البيانات الوصفية والنصوص الكاملة | https://pmc.ncbi.nlm.nih.gov/api/oai/v1/mh/ |
| **BioC API** | النصوص الكاملة بصيغة XML أو JSON | https://www.ncbi.nlm.nih.gov/research/bionlp/RESTful/pmcoa.cgi |
| **PMC ID Converter** | تحويل بين معرّفات المقالات | https://pmc.ncbi.nlm.nih.gov/tools/idconv/api/v1/articles/ |
| **E-Utilities** | واجهة موحدة لجميع قواعد بيانات NCBI | https://www.ncbi.nlm.nih.gov/books/NBK25499/ |

### 4. PubMed
**الموقع:** https://pubmed.ncbi.nlm.nih.gov/
**الوصف:** قاعدة بيانات المكتبة الوطنية للطب تحتوي على الاستشهادات والملخصات
**المميزات:**
- أكثر من 39 مليون استشهاد من الأدبيات الطبية
- بيانات متاحة عبر FTP والـ E-utilities API
- بحث مجاني بدون تسجيل

## الخصائص المهمة للاستخدام في RAG

### المميزات المشتركة
1. **مفتوحة الوصول:** جميع المصادر متاحة بدون رسوم
2. **موثوقة:** من مؤسسات حكومية أمريكية معروفة (NIH, NLM)
3. **محدثة:** البيانات تُحدث بانتظام
4. **واجهات برمجية:** توفر APIs للوصول الآلي
5. **نصوص كاملة:** توفر محتوى نصي كامل وليس ملخصات فقط

### القيود والاعتبارات
1. **حقوق الطبع والنشر:** يجب الالتزام بشروط الترخيص لكل مقالة
2. **الاستخدام المسموح:** يُسمح فقط بالاسترجاع الآلي عبر الخدمات المحددة
3. **المحتوى:** ليس جميع المقالات متاحة للتعدين النصي

## الخطة الموصى بها

### للبيانات الأولية
